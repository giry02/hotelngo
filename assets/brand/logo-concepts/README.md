# HotelNGo 로고 콘셉트

특수문자 없이 `HotelNGo`를 유지하고, `n`을 호텔과 여행을 연결하는 핵심 심볼로 설계한 1차 시안이다.

## 시안

1. `01-gate-n.svg` — 열린 게이트 형태의 n. 가장 단순하고 파비콘·앱 아이콘 확장이 쉽다.
2. `02-bridge-n.svg` — Hotel과 Go를 잇는 브리지. 플랫폼/연동 의미가 강하다.
3. `03-route-n.svg` — n 내부의 이동 경로. 예약 이후 여행으로 진행하는 인상이 강하다.
4. `04-warm-portal-n.svg` — 따뜻한 포털/현관. 숙박과 환대의 인상이 강하다.

## 색상

- Gate navy: `#123E69`
- Deep teal: `#0B5351`
- Connection orange: `#FF6B2C`
- Warm cream: `#FFF2DF`

## 제작 상태

- 모든 파일은 확대 가능한 SVG다.
- `n` 심볼과 도형은 벡터 패스로 작성했다.
- 1차 비교를 위해 워드마크 글자는 라이브 SVG 텍스트다. 최종안 선정 후 글자 간격과 형태를 보정하고 전체 문자를 패스로 변환한다.
- 상표·도메인 사용 가능성 검토는 별도 절차다.
