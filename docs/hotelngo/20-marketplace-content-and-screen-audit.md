# HotelnGo 현지 업체 콘텐츠·화면 점검

점검일: 2026-07-24
점검 기준: 로컬 정적 프로토타입 + JSON Mock

## 이번 보완 범위

- 현지 업체 목록의 업종별 대표 이미지를 분리했다.
- 공통 업체 상세에 실제 고객이 확인해야 할 주소, 운영시간, 문의 방법, 지원 언어, 예약 방식, 주변 이동 정보를 우선 배치했다.
- 업체별 소개, 주요 특징, 시설·서비스, 이용 정책을 `place-profiles.json`에서 표시한다.
- 호텔은 `hotel-content.json`의 객실 사진, 정원, 요금, 판매 가능 수량을 표시한다.
- 골프, 차량, 음식점, 스파, 투어는 `marketplace-catalog.json`의 상품, 사진, 포함 사항, 가격, 시간 슬롯을 표시한다.
- 내부 관리 방식, 소유권, 정보 출처는 고객용 핵심 정보 아래의 출처·운영권 영역으로 이동했다.
- 메인 히어로 캐러셀 버튼을 검색창 위 안전영역으로 이동했다.

화면은 JSON 경계를 통해서만 데이터를 읽는다. 실제 연동 단계에서는 같은 응답 형태를 유지하고 Mock API를 서버 API로 교체한다.

## 실제 서비스 전 남은 경계

- 현재 업체 정보, 가격, 슬롯은 JSON Mock이며 실시간 공급자 응답이 아니다.
- 호텔 객실 요금·재고는 이후 Hotel_PMS `/api/v1/channel/*` 응답으로 교체해야 한다.
- 다른 업종은 업체용 상품·재고 API와 예약 승인·취소·정산 API가 필요하다.
- 견적/요청형 상품은 업체 승인 전 `CONFIRMED`로 표시하면 안 된다.
- 결제, 환불, 정산, 실시간 알림은 서버와 결제대행 연동 후 최종 검증해야 한다.
